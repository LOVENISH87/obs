
## Correct movement patterns (memorize this table)

| Goal           | Movement           |
| -------------- | ------------------ |
| Adjacent pairs | `i += 2`, `j += 2` |
| Full reverse   | `i++`, `j--`       |
| Sliding window | `i++`, `j++`       |
| Skip one       | `i++`, `j += 2`    |
